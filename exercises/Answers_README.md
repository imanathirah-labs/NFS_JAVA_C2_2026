# Why is CourseOffering more useful than using only Course when building a real web application?
- `Course` is the generic course template (name, duration, level).
- `CourseOffering` is a specific scheduled run of that course with dates, capacity, instructor, and delivery mode.
- The same course can have many offerings, so `CourseOffering` lets you separate the template from individual sessions.
- This makes it easier to manage schedules, enrollments, instructors, and availability without duplicating course data.
